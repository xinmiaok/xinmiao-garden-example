description :: 在统计学中，线性回归（Linear Regression）是利用称为[[线性回归方程]]的[[最小平方函数]]对一个或多个自变量和因变量之间关系进行建模的一种回归分析。这种函数是一个或多个称为[[回归系数]]的模型参数的线性组合。只有一个自变量的情况称为简单回归，大于一个自变量情况的叫做多元回归。（这反过来又应当由多个相关的因变量预测的多元线性回归区别，而不是一个单一的标量变量。）

用于:: 数学, 趋势线, 流行病学, 金融, 经济学
